# Scope — T-PERF-01

## In scope
- Модификация или создание `router_performance_benchmark_SUITE` для обеспечения корректных замеров в real-nats режиме.
- Скрипт `scripts/bench_router.sh` для запуска и парсинга результатов.
- Генерация JSON и Markdown отчетов.
- Автоматический запуск/остановка NATS в рамках скрипта.
- Поддержка различных уровней нагрузки (RequestCount).

## Out of scope
- Масштабные стресс-тесты (>10 минут).
- Профилирование CPU (fprof/eflame) в рамках этой задачи.
- Бенчмаркинг gRPC overhead (фокус на ядре Router).

## Constraints
- Результаты должны быть воспроизводимы на локальной машине разработчика.
- Артефакты логов и отчетов всегда сохраняются в `_artifacts/`.
