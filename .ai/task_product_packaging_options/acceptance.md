# Acceptance — T-PRODUCT-01

## Done criteria
1. Есть один canonical doc (например `docs/PRODUCT_POSITIONING.md`) с 3 вариантами.
2. Для каждого варианта:
   * кто покупатель
   * как ставится/интегрируется
   * какие гарантии (и ссылка на SLO gate)
   * какие ограничения
3. Выбран “primary packaging” и зафиксировано решение.
4. Есть “elevator pitch” (5–7 строк) и “integration narrative” (Gateway/CAF).
