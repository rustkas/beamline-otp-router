## Acceptance Criteria

- [ ] Прогон бенчмарка автоматизирован скриптом
- [ ] NATS поднимается и останавливается автоматически
- [ ] Baseline зафиксирован в JSON (perf/baseline_cp1.json)
- [ ] Summary содержит численные значения (не "ok/failed")
- [ ] Артефакты сохраняются в `_artifacts/`
- [ ] Повторный запуск даёт сопоставимые результаты (±20%)
- [ ] Gate script проверяет регрессии

PASS ≠ "быстро".
PASS = "измерено, зафиксировано, воспроизводимо".
